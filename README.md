# Advent of Code

## Setup

Create a new day:
```bash
./newday.sh <year> <day>
```

This creates:
- `<year>/day<##>/solution.cpp` - solution template
- `<year>/day<##>/problem.md` - copy problem description here
- `<year>/day<##>/input.txt` - copy puzzle input here

## Compile and Run

```bash
make
./aoc <year> <day>
```
